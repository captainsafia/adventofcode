## Overview

My solutions for [http://adventofcode.com/](http://adventofcode.com/) in C# using .NET Core.

## Building and Testing

To build and test the puzzles run one of the following commands:

**Linux/OS X**

```sh
./build.sh
```

**Windows**

```powershell
.\Build.ps1
```
